# partner-wms-labels
ระบบสร้าง Barcode มาตรฐานกลาง (Data Pattern) สำหรับเครือข่าย Partner เพื่อใช้ในการรับ-ส่งสินค้าในระบบ WMS ร่วมกัน

# 📊 มาตรฐานโครงสร้างข้อมูล (Data Pattern)
เพื่อให้ระบบ WMS/ERP ของทุกองค์กรสามารถสแกนและอ่านข้อมูลได้ตรงกัน ข้อมูลใน Barcode จะถูกคั่นด้วยเครื่องหมาย `|` (Pipe) ตามลำดับดังนี้:
*   **วัตถุดิบ (RM):** `Part | Lot | Qty | Rolls | Unit | PO`
*   **สินค้าสำเร็จรูป (FG):** `Part | Lot | Qty | Rolls | Unit | Pack | M/C | PO`

# 🖨️ รูปแบบการพิมพ์ (Print Modes)
ระบบรองรับการพิมพ์ 2 รูปแบบ เพื่อความยืดหยุ่นหน้างาน:
1.  **✂️ พิมพ์แยกตามม้วน (Split Mode):** ระบบจะนำ "Qty รวม" มาหารเฉลี่ยตามจำนวนม้วนอัตโนมัติ และรันเลข Lot ย่อย (-R1, -R2...) เหมาะสำหรับติดรายชิ้น[cite: 1]
2.  **📦 พิมพ์ใบรวม (Single Mode):** พิมพ์ Label เพียง 1 ใบครอบคลุมยอดรวมทั้งหมด เหมาะสำหรับติดพาเลทหรือกล่องใหญ่[cite: 1]
