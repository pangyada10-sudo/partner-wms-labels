🔗 แหล่งข้อมูลสินค้า (Google Sheets)

วางลิงก์ Published CSV ของ Google Sheets ที่เป็น Master Data สินค้า — ข้อมูลจะถูกโหลดเข้าระบบและใช้ใน Autocomplete โดยผู้ใช้งาน (Partner) จะไม่เห็นรายการทั้งหมด แค่ค้นหาได้เท่านั้น

— ยังไม่ได้ตั้งค่า
📋 รูปแบบ Google Sheets (Column A→E)
A: Part No.  B: Product Name  C: Type (RM/FG)  D: Unit  E: Note (optional)
Row 1 = Header (ข้ามอัตโนมัติ) · ตัวอย่าง: AAP050C400, A-PET CLEAR 0.50mm*400mm, RM, Kgs
วิธี Publish Google Sheets เป็น CSV:
File → Share → Publish to web → เลือก Sheet → เลือก "Comma-separated values (.csv)" → Publish → คัดลอก URL มาวางด้านบน

💾 สถานะข้อมูลในระบบ

ข้อมูลสินค้าที่โหลดจาก Google Sheets จะถูก Cache ไว้ในเครื่อง ใช้ได้แม้ออฟไลน์ชั่วคราว

📊 มาตรฐาน Barcode / QR Data String (WMS)

ทั้ง Barcode และ QR Code ใช้ Data String เดียวกัน — คั่นด้วย | 7 ท่อน:

Part | Lot | Qty | Unit | Rolls | Ref | PO

*ท้ายที่ไม่มีข้อมูลจะถูกตัดทิ้งอัตโนมัติ เพื่อให้ Barcode สั้น สแกนง่ายขึ้น

📡 การเลือก Barcode vs QR Code

Barcode (CODE128 / 1D) — เหมาะสำหรับ Data String สั้น สแกนเร็ว ใช้กับ Scanner ธรรมดา
QR Code (2D) — รองรับ Data String ยาวมากขึ้น เหมาะเมื่อมีข้อมูลหลายช่อง Scanner ทั่วไปที่รองรับ 2D สแกนได้
Scanner 1D/2D ของบริษัทสามารถสแกนได้ทั้งสองรูปแบบ

🔗 วิธีเชื่อม Google Sheets

ไปที่แท็บ ⚙️ ตั้งค่าระบบ → วาง URL ของ Google Sheets ที่ Publish เป็น CSV แล้ว → กด "บันทึกและโหลดข้อมูล"
ระบบจะดึงข้อมูลสินค้าโดยอัตโนมัติ ผู้ใช้งาน (Partner) จะค้นหาสินค้าได้ แต่ไม่เห็นรายการทั้งหมด
